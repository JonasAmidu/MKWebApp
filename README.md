# MKWebApp

A React web application built with Create React App, hosted and deployed via AWS Amplify.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 18 |
| Build Tool | Create React App |
| Hosting | AWS Amplify |
| State | React hooks |
| Styling | CSS Modules / styled-components |

## Local Setup

```bash
# Clone the repo
git clone https://github.com/JonasAmidu/MKWebApp.git
cd MKWebApp

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm start` | Runs the app in development mode |
| `npm test` | Launches the test runner in watch mode |
| `npm run build` | Builds the app for production to `build/` |
| `npm run eject` | One-way eject from CRA configuration |
